NPM Package

CSS custom properties and Tailwind v4 theme from Figma design tokens.

@hferello/design-libraries-npm
View on GitHub

A Style Dictionary v4 pipeline that transforms Figma-exported DTCG token JSON into alias-preserving CSS custom properties and a Tailwind v4 @theme configuration. Each library produces three CSS files — use the one that fits your stack.

Install
# GitHub Packages registry npm install @hferello/design-libraries-npm --registry=https://npm.pkg.github.com
Usage
/* Tailwind v4 project */ @import "@hferello/design-libraries-npm/golden-ratio/tailwind-theme.css"; /* Non-Tailwind project */ @import "@hferello/design-libraries-npm/golden-ratio/tokens.css";
<!-- Tailwind utility examples --> <div class="p-tight gap-section rounded-soft blur-void"> <p class="text-base text-fg-neutral-aa bg-neutral-faint">Hello</p> </div>

Output files
File Purpose
primitives.css Raw palette values in :root. Number scale, all colour hues, transparent overlays.
tokens.css Semantic layer for non-Tailwind consumers. Imports primitives, defines var(--primitive-*) aliases, .dark overrides, and @media responsive breakpoints.
tailwind-theme.css Tailwind v4 entry point. Splits tokens into @theme static (utility classes) and :root (plain CSS variables).

Pipeline
Source (10 DTCG JSON files per library) │ ▼ Style Dictionary v4 ├─ Parser → namespace wrapping ├─ Preprocessor → flatten Figma colours to hex/rgba ├─ Name transform → static mappings + namespace routing ├─ Value transform→ px units, opacity normalisation └─ Alias transform→ aliasData → var() references │ ▼ Output (3 CSS files per library) ├─ primitives.css ├─ tokens.css └─ tailwind-theme.css
